Парковки

Парковки

Оценка доступности:
Доступно
100%
На территории вокзального комплекса расположены 2 автопарковки. 1 парковка расположена с восточной стороны вокзала. 2 парковка расположена между 2 и 3 павильонами входного контроля. Обе парковки расположены на расстоянии 30м от входа в здание вокзала. Места для стоянки автотранспортных средств инвалидов обозначены дорожным знаком, разметкой.
Привокзальная площадь

Привокзальная площадь

Оценка доступности:
Доступно
100%
Обеспечена достаточная ширина проходов на путях движения маломобильных пассажиров, имеется пространство для разворота кресла-коляски. Пути движения по прилегающей территории выполнены с понижением бордюрного пандуса. Ширина пути пешеходного движения не менее 2м.
Входные группы

Входные группы

Оценка доступности:
Доступно
100%
Вход в здание вокзала со стороны города осуществляется через досмотровую зону. В зоне досмотра для пассажиров с кардиостимуляторами оборудован отдельный проход. У всех входных групп в здание вокзала установлены кнопки вызова персонала. Вход со стороны платформ не имеет вертикальных препятствий. На платформах установлены кнопки экстренного вызова. Ширина дверных проемов более 0,9м. Входные группы вокзала автоматические с фотоэлементом.
Пути движения

Пути движения

Оценка доступности:
Доступно
100%
Места перепада высот оборудованы пандусами. Обеспечена достаточная ширина проходов на путях движения маломобильных пассажиров, имеется пространство для разворота кресла-коляски. Кнопка вызова персонала размещена при входе в здание вокзала. В здании вокзала размещены информационные тактильные таблички для людей с нарушением зрения. На входах в здание вокзала установлены мнемосхемы. На стеклянные поверхности нанесены контрастные знаки, нижние и верхние ступени лестничных маршей имеют контрастные полосы.

100%

Ширина дверных проемов не менее 0,9м.

Кассы

Кассы

Оценка доступности:
Частично доступно
87.5%
Кассовый блок пригородного сообщений с пониженным прилавком для инвалидов-колясочников. Установлено индукционное оборудование для пассажиров с нарушением слуха. Кассы дальнего следования оборудованы установкой "электронная очередь". Информация и навигация

Информация и навигация

Оценка доступности:
Доступно
100%

Средства транспортировки

Средства транспортировки

Оценка доступности:
Доступно
100%
Доступ маломобильных пассажиров осуществляется на все этажи здания вокзала. Для подъема по лестницам инвалидов на креслах-колясках имеются мобильные подъемники ( 3 шт.) , кнопки вызова работнтков вокзала размещены рядом с ними, а также имеются лифты соединяющие все этажи. При выходе на перрон с восточной и западной стороны установлены стационарные подъемные платформы. При выходе из тоннеля №2 на пассажирские платформы установлены 8 лифтов при подъеме на каждую пассажирскую платформу. Для транспортировки по вокзальному комплексу имеется кресло-коляска, предоставляется при обращении к дежурному по вокзалу.
