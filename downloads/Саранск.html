Привокзальная площадь

Привокзальная площадь

Оценка доступности:
Доступно
100%
Обеспечена достаточная ширина проходов на путях движения маломобильных пассажиров, имеется пространство для разворота кресла-коляски.
Входные группы

Входные группы

Оценка доступности:
Доступно
100%
Вход на территорию вокзального комплекса со стороны города через досмотровый павильон. Входа в здание вокзала с обеих сторон оборудованы пандусами, вход со стороны платформ не имеет вертикальных препятствий. С обеих сторон пандуса установлены поручни, ширина между поручнями пандуса более 0,9м. На входных группах установлены доводчики, обеспечивающие легкое открытие дверей и плавное закрытие. Ширина дверных проемов более 0,9м.
Пути движения

Пути движения

Оценка доступности:
Доступно
100%
Обеспечена достаточная ширина проходов на путях движения маломобильных пассажиров, имеется пространство для разворота кресла-коляски. Для ориентирования людей с нарушением зрения предусмотрены тактильные напольные указатели. Кнопки вызова персонала размещены у правого входа в здание вокзала.
Туалетные комнаты

Туалетные комнаты

Оценка доступности:
Доступно
100%
На 0-ом этаже здания вокзала расположена адаптированная санитарная комнаты, оборудованная поручнями, кнопкой вызова персонала. На 2-ом этаже вокзала расположена санитарная комната с душевой кабиной для маломобильных пассажиров, оборудованная поручнями, кнопкой вызова персонала. Душевая кабина не имеет порогов, оборудована складным сидением, ручным душем, настенными поручнями. Ширина дверных проемов не менее 0,9м.
Залы ожидания

Залы ожидания

Оценка доступности:
Доступно
100%
В зале ожидания на 1-ом этаже вокзала расположена зона ожидания для маломобильных пассажиров, обозначена знаком доступности, оборудована индукционной петлей для пассажиров с нарушениями слуха, электронным табло. На 2-ом этаже вокзала расположен бизнес - зал с адаптированной санитарной комнатой, оборудованной поручнями и кнопкой вызова персонала. Ширина дверных проемов менее 0,9м.

Комнаты отдыха

Комнаты отдыха

Оценка доступности:
Доступно
100%
На 2-м этаже вокзала адаптирована комната отдыха для маломобильных пассажиров. Санитарная комната оборудована в соседнем помещении, оснащена душем, складным сидением, поручнями, кнопкой вызова персонала. Ширина дверных проемов не менее 0,9м. Для обеспечения доступа с 1-го этажа на 2-й предусмотрен лифт адаптированный для маломобильных пассажиров.
Платформы

Платформы

Оценка доступности:
Доступно
100%
На выходе к платформам имеется доступный проход для пассажиров на креслах - колясках. На перроне для ориентирования людей с нарушением зрения предусмотрены тактильные указатели со стороны посадки/высадки пассажиров в вагон. На платформах границы опасной зоны обозначены сигнальной полосой. Предусмотрено дублирование визуальной информации речевой (звуковой).

Коммерческие помещения

Входные группы адаптированы для маломобильных пассажиров, в зонах приема пищи имеются прилавки, столики высотой от уровня пола не более 0,85м.
Информация и навигация

Информация и навигация

Оценка доступности:
Доступно
100%
Имеется визуальная информация и навигация. Звуковая информация дублируется визуальной на табло. Информирование пассажиров по громкоговорящей связи. Тактильные указатели для ориентирования пассажиров с нарушениями зрения на привокзальной территории и в здании вокзала отсутствуют. Организовано сопровождение маломобильных пассажиров на вокзале по предварительной заявке или при личном обращении к работникам вокзала.
Средства транспортировки

Средства транспортировки

Оценка доступности:
Доступно
100%
Для транспортировки по вокзальному комплексу маломобильных пассажиров имеются кресло- коляска, лестничный подъемник (для спуска на 0-й этаж), вертикальный подъемник (для подъема на 2-й этаж). Для посадки/высадки с низкой платформы в поезд имеется вертикальная подъемная платформа. Оборудование предоставляется при обращении к дежурному персоналу вокзала. Пешеходный мост оборудован лифтами.

Оформить заявку на оказание услуг помощи и сопровождения можно круглосуточно без выходных дней по бесплатному федеральному номеру Центра содействия мобильности ОАО "РЖД" 8(800)775-00-00 (доб. 1).
