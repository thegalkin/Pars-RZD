Парковки

Парковки

Оценка доступности:

Привокзальная площадь

Привокзальная площадь

Оценка доступности:
Частично доступно
87.5%
Обеспечена достаточная ширина проходов на путях движения маломобильных пассажиров, имеется пространство для разворота кресла-коляски.
Входные группы

Входные группы

Оценка доступности:
Частично доступно
87.5%
Вход в здание вокзала со стороны города осуществляется через досмотровый павильон, вход со стороны платформ оборудован пандусом. С обеих сторон пандуса установлены поручни. Ширина дверных проемов более 0,9м. У входа в досмотровый павильон установлена кнопка вызова персонала.
Пути движения

Пути движения

Оценка доступности:
Частично доступно
75%
Обеспечена достаточная ширина проходов на путях движения маломобильных пассажиров, имеется пространство для разворота кресла-коляски. Кнопка вызова персонала размещена при входе в досмотровый павильон.

Оценка доступности:
Частично доступно

Платформы

Платформы

Оценка доступности:
Частично доступно
87.5%
На выходе к платформам имеется доступный проход для пассажиров на креслах - колясках. На перроне для ориентирования людей с нарушением зрения предусмотрены тактильные указатели со стороны посадки/высадки пассажиров в вагон. На платформах границы опасной зоны обозначены сигнальной полосой. Проход на платформу №2, №3 осуществляется по пешеходным настилам с перрона. Предусмотрено дублирование визуальной информации речевой (звуковой).

Информация и навигация

Информация и навигация

Оценка доступности:
Частично доступно
75%
Имеется визуальная информация и навигация. Информирование пассажиров по громкоговорящей связи. Тактильные средства навигации для людей с нарушениями зрения отсутствуют. Организовано сопровождение маломобильных пассажиров на вокзале по предварительной заявке или при личном обращении к работникам вокзала.
Средства транспортировки

Средства транспортировки

Оценка доступности:
Доступно
100%
Для транспортировки по вокзальному комплексу имеется кресло-коляска, предоставляется при обращении к дежурному персоналу вокзала. Для посадки/высадки с низкой платформы в поезд имеется вертикальная подъемная платформа.

Оформить заявку на оказание услуг помощи и сопровождения можно круглосуточно без выходных дней по бесплатному федеральному номеру Центра содействия мобильности ОАО "РЖД" 8(800)775-00-00 (доб. 1).
